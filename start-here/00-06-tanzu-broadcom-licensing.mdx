# Tanzu / Broadcom Licensing

## Open-source vs. commercial

RabbitMQ is open-source under the **Mozilla Public License 2.0 (MPL-2.0)**. The open-source version is free to use with community support.

Broadcom offers a commercial distribution called **VMware Tanzu RabbitMQ** (formerly Pivotal RabbitMQ) that adds:
- Enterprise features (JMS queues, delayed queues, enhanced LDAP)
- Broadcom support with defined SLAs
- Extended version support beyond community EOL
- Compliance documentation and CVE response SLAs

## What changes after the Broadcom acquisition

Broadcom acquired VMware in 2023. Key impacts:
- Licensing model shifted; enterprise features require Broadcom subscription
- RabbitMQ core remains open-source (MPL-2.0)
- Tanzu Application Platform bundles include RabbitMQ
- Support contracts now go through Broadcom channels

## AceMQ's role

AceMQ is an authorized Broadcom partner that can:
- Provide commercial RabbitMQ licensing
- Offer support under AceMQ SLAs (with Broadcom escalation path for Embedded tier)
- Assist with Tanzu migration and licensing assessment

Contact AceMQ for licensing questions before approaching Broadcom directly.
